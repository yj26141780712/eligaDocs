# 使用说明
在接口请求头中添加lang属性

#### 接口调用请求说明

>http请求方式：GET url

#### 请求头

| 参数  | 说明                                   |
|------|-----------------------------------------|
| lang |    本地化或者国际化使用的语言([语言说明](/api/lang.html))            |


#### 返回说明

```json
{
    "code":"200",
    "message":"ok",
    "obj":{
       "id",
       "name":"分段计时",  // 原始值
       "nameTr":"Timing System", // lang对应的翻译值,
       "desc":"",
       "descTr":""
    }
}
```
需要国际化的字段都会多一个字段名+Tr的字段
